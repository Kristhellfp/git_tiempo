Número Comando Descripción
1 Git clone Crea una copia de repositorio
2 Git Status Muestra el estado del repositorio
3 Git add Añade archivos al área de preparación
4 Git commit Captura una instantánea de los cambios preparados en ese momento del proyecto.
5 Git push Se utiliza para cargar cambios locales en un repositorio remoto.
6 Git pull Utilizado para actualizar la versión local de un repositorio desde otro remoto.
7 Git branch Permite crear una rama nueva.
8 Git checkout Permite desplazarte entre las ramas creadas por git branch.
9 Git merge Se utiliza para fusionar uno o más ramas dentro de la rama que tienes activa.
10 Git log Muestra el historial de confirmaciones (commits)
